What is this?

This is a learning platform where you can follow through activities and games to help learn across various topics around the information technology space.

This site is currently maintained by myself (Lewis Parmenter), but if you are interested in making changes/improvements to this content you can open a pull-request on the GitHub repository: https://github.com/lewpar/cyberbilby-learn

Now what?

You can select any topic from the menu on the left side of the screen and begin learning!
